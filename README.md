🌤️ SkyCast - Real-Time Weather Forecasting App
A sleek, data-driven web application that provides instant weather updates for any city worldwide. Built to demonstrate mastery over Dynamic UI updates and Third-Party API integration.

🚀 Project Overview
SkyCast connects to a global weather engine to deliver live temperature, humidity, wind speed, and atmospheric conditions. The app features a search-based interface with a focus on speed and visual accuracy.

🛠️ Tech Stack
Front-End: React.js / Vanilla JavaScript

API: OpenWeatherMap API / WeatherStack

Styling: CSS3 (Glassmorphism UI) / Tailwind CSS

Geolocating: Browser Geolocation API (for local weather)

Icons: Skycons / FontAwesome

✨ Key Features
Global Search: Fetch weather data for over 200,000+ cities instantly.

Smart Backgrounds: The UI theme changes dynamically based on the current weather condition (e.g., blue for clear sky, grey for clouds).

Metric/Imperial Toggle: Switch between Celsius and Fahrenheit with a single click.

Auto-Location: Uses the user's GPS coordinates to provide a "Weather at your current location" feature.

Detailed Metrics: Includes humidity, UV index, and "Feels Like" temperatures.

🏗️ Technical Challenges Solved
Unit Conversion: Implemented mathematical formulas to convert Kelvin (default API output) into Celsius and Fahrenheit.

Conditional Logic: Built a robust mapping system to match weather "Condition Codes" with specific background images and icons.
